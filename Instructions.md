## Overview
You will build a Python Flask web application that leverages the Groq API to summarize text content from an uploaded .TXT file. This application will also offer a user-friendly interface with robust backend fuctionality that clearly indicate and seperate areas for file upload, display of the original content, and summarized output so the workflow is easy to understand.
